Diagnostic Modelling of Coaxial Cable Network
May 2012 - Aug 2012

This project was done during the summer of 2012 as an assistant researcher in the Electromagnetics group of University of Toronto. The objective was to design an algorithm to model a coaxial cable network using solely reflection measurements. Application of this algorithm allows technicians who often only has access to one port of a network to compute the physical dimensions of the entire network. The behavior of reflected signals were identified using network analyzer. Its processing and the mathematic modelling of cable network was done using Matlab.
